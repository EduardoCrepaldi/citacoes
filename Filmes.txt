Aprendendo GitHub

echo ~ -> Mostra a pasta Pessoal

cd ~/PastaOndeVoceQuer

git init ->ira criar um repositorio Vazio

git add conteudo.txt -> Comando para que o arquivo seja rastreado

git status -> Podemos ver a situação dos arquivos no repositorio

git commit -m "Nome para Salvar " -> Comando para gravar arquivo no repositorio (-m informa a mensagem do commit)

git log -> Verifica o historico de alterações
